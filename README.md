## Description
"Brain Games" is a set of logic games:
  + "Even Number"
  + "Calculator"
  + "Greatest Common Divisor"
  + "Arithmetic Progression"
  + "Prime Number"
## Requirements
  + OS Linux
  + Python >= 3.10
## Installation
```
python3 -m pip install --user dist/*.whl
```
